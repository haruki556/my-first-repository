# weight-tracker

自分専用の体重記録Webアプリです。  
ブラウザだけで動作し、データは `localStorage` に保存されます。

## 使い方

1. `index.html` をブラウザで開く
2. 日付と体重を入力して保存
3. 一覧・7日平均・推移グラフを確認
4. 必要なら `CSV出力` でバックアップ
5. バックアップを戻したい場合は、`CSVインポート` からCSVを選択
6. 日付を切り替えると、その日の入力欄が自動で反映されます（なければ空になります）
7. `全データ削除` はローカル保存データ（このブラウザ）を消します

## CSV形式

エクスポートされるCSVは `date,weight,note` の順です（ヘッダ付き）。  
インポート時は同じ日付の既存データが確認のうえ上書きされます。  
また、CSV内に同じ日付が複数ある場合は「最後の行」を採用します。

- Excelなどが吐くCSVにBOMが含まれていても、できるだけ読み込みます

## 補足

- 保存先はブラウザ内のため、別ブラウザや別端末には自動同期されません
- 同じ端末でもブラウザデータ削除時に記録が消える可能性があります
- 長期保存したい場合は定期的にCSVを出力してください

## ローカルでサーバ起動（推奨）

ブラウザで直接開いても動きますが、同じネットワーク内で開きたい場合などはサーバ経由が便利です。

```sh
cd weight-tracker
python3 -m http.server 8080
```

その後、ブラウザで `http://localhost:8080/` を開いてください。
